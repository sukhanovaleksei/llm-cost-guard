# Runnable examples

All examples run directly from the package sources and check the current `src` version, not the published npm build.

## Core
- `npm run example:core:basic-run`
- `npm run example:core:basic-run-blocked`
- `npm run example:core:basic-run-downgraded`

## Adapters
- `npm run example:adapter:openai`
- `npm run example:adapter:anthropic`

## Budgets
- `npm run example:budget:per-user`

## Storage
- `npm run example:storage:redis`

## Analytics
- `npm run example:analytics:spike`

## Framework helpers
- `npm run example:framework:express`
- `npm run example:framework:next`
- `npm run example:framework:nest`

## Typecheck
- `npm run examples:typecheck`

## Smoke
- `npm run examples:smoke`
